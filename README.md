# Smart Apps Student Demo

Dokumentationsgetriebenes Demo-Projekt fuer BWL-Studierende, die mit KI-Unterstuetzung Programmieren und Smart Apps entwerfen lernen.

## Einstieg

- `docs/spec.md` ist die zentrale fachliche Quelle und ersetzt eine klassische PRD.
- `docs/backlog.md` enthaelt Feature-Hypothesen und Uebungsaufgaben mit stabilen `SAD-NNN` IDs.
- `docs/architecture.md` beschreibt eine moegliche technische Zielstruktur.
- `docs/decisions.md` dokumentiert relevante Produkt- und Architekturentscheidungen.
- `CLAUDE.md` und `AGENTS.md` briefen AI-Agenten fuer die weitere Arbeit.

## Beispielprojekt

Die aktuelle Spec beschreibt den "Campus Cafe Manager": eine kleine Smart App fuer ein fiktives Campus-Cafe. Studierende koennen daran Umsatzdaten auswerten, Lagerbestaende planen, Aktionsideen simulieren und einfache Management-Entscheidungen mit KI-Unterstuetzung vorbereiten.
